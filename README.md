# Personality Test Website

This is a simple and fun personality test website designed for a young audience, particularly elementary school students. It features a series of engaging questions with a space/cosmic theme, leading to a playful personality result.

## Features

- **Start Screen:** A welcoming screen with a title and a "Start" button.
- **Question Series:** A set of four multiple-choice questions displayed one by one.
- **Progress Bar:** A visual indicator to show the user their progress through the test.
- **Dynamic Results:** Based on the user's answers, the site calculates and displays one of three personality types:
  - **The Dazzling Comet:** For energetic and creative kids.
  - **The Wise Nebula:** For thoughtful and curious minds.
  - **The Friendly Galaxy:** For social and friendly personalities.
- **Result Page:** Shows the user's personality title, a descriptive paragraph, a relevant image, and a "Try Again" button.
- **Responsive Design:** The layout is fully responsive and works well on both desktop and mobile devices.

## Design

The website uses a modern and bold design with a cosmic theme:

- **Color Palette:** A deep blue and purple background creates a space-like atmosphere, with vibrant pink and yellow for interactive elements like buttons and titles.
- **Typography:** A clean, readable sans-serif font is used, with large, expressive headings.
- **Animations:** Smooth fade-in animations for content transitions to create a polished user experience.
- **Layout:** A centered, single-column layout ensures focus and usability.

## How to Run

1.  Open `index.html` in a web browser.
2.  Click the "Start the Test" button.
3.  Answer the questions.
4.  View your result!

This project was built using modern web standards with HTML, CSS, and JavaScript, and it does not require any external frameworks or libraries.
